# Twis Verification

Sistema de verificación con CAPTCHA para Discord hecho en BDFD.

«“Del caos a la claridad.” — Twis Spark»

## ✨ Características

- CAPTCHA interactivo
- Verificación automática
- Mensajes modernos con Containers
- Protección contra múltiples verificaciones
- Sistema rápido y fácil de configurar
- Compatible con roles de verificación
- Diseño limpio y moderno
- Fácil de modificar y expandir
- Optimizado para servidores pequeños y grandes

## 📦 Funciones

- Verifica usuarios mediante CAPTCHA
- Detecta usuarios ya verificados
- Previene spam de verificaciones
- Mensajes personalizados
- Diseño moderno usando Components V2
- Fácil integración en cualquier servidor
- Configuración rápida

---

## 🚀 Instalación

1. Importa los comandos en tu bot.
2. Configura el rol de verificación.
3. Busca el canal de verificación.
4. Ejecuta el sistema.
5. Personaliza los mensajes si quieres.

---

🛠️ Tecnologías

- BDScript
- Components V2

## 💡 Objetivo

Twis Verification fue creado para ofrecer un sistema de verificación moderno, limpio y fácil de usar sin configuraciones absurdamente complicadas.

La idea no es llenar el servidor de comandos innecesarios.

La idea es simple:

«seguridad rápida, clara y moderna.»
